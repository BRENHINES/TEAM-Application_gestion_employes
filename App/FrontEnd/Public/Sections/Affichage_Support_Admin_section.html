Support

Il s'agit du support de manipulation de cette application.
Cette application vous permet, (à l'administrateur) de gérer les employés de son entreprise.
Comme fonction principale, nous avons l' authentification et la gestion de rôle avec également la gestion des différentes données de l'utilisateur.
Elle vous permet entre autre de :
  • Créer des employés,
  • Modifier les employés,
  • Supprimer les employés.
Mais également de gérer les congés de manière efficiente avec un système qui permette de demander des congés (créer), de Modifier sa demande, de suppression de congés; Mais également pour l'administrateur de valider lesdits congés.
En outre, elle permet de gérer les réunions de l'entreprise, en les affichant sur le dashboard de chaque utilisateur.
Cette application a été coder avec ElectronJS, SASS, JS,AJAX pour le Frontend ; Django, MySQL pour le Backend et Git, Docker pour le versionning et le déploiement.
